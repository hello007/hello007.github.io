🌟 Level 1: easy-vibe - AI 编程教育课程（零基础友好）

GitHub 仓库：datawhalechina/easy-vibe 难度等级：⭐☆☆☆☆（适合所有人，无需编程基础） 项目简介：

把想法变成现实，就是这么简单 | Turn ideas into reality. It's that simple.

核心理念：

不需要学编程，只要会说话，就能做出你的第一个 AI 应用

特色功能：
  • 🎮 新手专属学习地图 - 零基础专属指引，清晰规划路径
  • 📝 手把手图文教程 - 保姆级图文详解，如同私教在旁
  • 🖱️ 沉浸式模拟编程 - 虚拟鼠标自动导览，快速上手 IDE
  • 🎨 看得见的 AI 原理 - 算法原理动画化，一眼看懂 AI
  • 🔄 像玩游戏一样学 RAG - 独家交互组件，点击看清数据流
  • 💻 可视化终端原理 - 命令行操作可视化
课程结构（四阶段渐进式）：

🎮 阶段 0 - 幼儿园（5分钟体验）

  • 学习内容：AI 编程初体验、贪吃蛇小游戏
  • 产出：5 分钟做出第一个 AI 应用
  • 适合人群：所有人

💡 阶段 1 - AI 产品经理（零基础）

  • 学习内容：AI IDE 工具、需求拆解、产品原型设计
  • 产出：完整的可演示产品原型
  • 适合人群：零基础/产品经理/创业者

🚀 阶段 2 - 初中级开发工程师

  • 学习内容：全栈开发、数据库、AI 集成、部署运维
  • 产出：可上线的全栈 AI 应用
  • 适合人群：开发者/进阶学习者

🦞 阶段 3 - 高级开发工程师

  • 学习内容：Claude Code 核心技能、跨平台开发（微信小程序、PWA、Electron 等）
  • 产出：跨平台 AI 应用
  • 适合人群：对 AI Agent 感兴趣的开发者
附录知识库：
  • 📚 9 大知识领域，80+ 交互式专题
  • 💻 计算机基础、🔧 开发工具、🌐 浏览器与前端
  • 🖥️ 服务器与后端、📊 数据、🏗️ 架构与系统设计
  • ☁️ 基础设施与运维、🤖 人工智能、🎯 工程素养
技术栈：
  • 文档系统：VitePress 2.0.0-alpha.15
  • Node.js：>= 18.0.0
  • 部署：Vercel
在线访问：https://datawhalechina.github.io/easy-vibe/ 本地启动：

npm run dev
# 访问 http://localhost:5173

🌟 Level 4: vibe-coding-cn - Vibe Coding 中文指南（方法论深入）

GitHub 仓库：tukuaiai/vibe-coding-cn 上游仓库：EnzeD/vibe-coding 难度等级：⭐⭐⭐⭐☆（适合有实践经验，想深入学习方法论的开发者） 项目简介：

一个通过与 AI 结对编程，将想法变为现实的终极工作站

核心理念：
  • 规划驱动 - 先规划，后编码
  • 模块化 - 避免单体巨文件
  • 索引构建 - 解决上下文窗口限制
  • 胶水编程 - 复用 > 原创，连接 > 创造
  • AI 蜂群协作 - 多 AI 协作，人从瓶颈变为调度者
三层架构：

📚 assets/documents/ - 知识层

  • principles/ - 原则与思想
  • fundamentals/ - 基础指南（胶水编程、血的教训、常见坑等）
  • philosophy/ - 哲学与方法论（23 种可落地工程方法）
  • guides/ - 入门与方法
  • getting-started/ - 入门指南（网络、IDE、CLI 配置）
  • playbook/ - 方法论实战（Canvas 白板、AI 蜂群等）
  • case-studies/ - 实战案例

💬 assets/prompts/ - 指令层

  • 系统提示词（约束 AI 行为边界）
  • 编程提示词（需求澄清、计划、执行）
  • 用户提示词（日常使用）
  • 在线表格：Google Sheets

⚡ assets/skills/ - 能力层

  • 可复用的 AI 技能库
  • 元技能：skills-skills（生成 Skills 的技能）
  • SOP 生成：sop-generator
  • Canvas 白板驱动：canvas-dev
实验性方法：

🔑 元方法论

  • α-提示词（生成器）：生成其他提示词或技能
  • Ω-提示词（优化器）：优化其他提示词或技能
  • 递归闭环：持续自我优化的 AI 系统

🧬 胶水编程 (Glue Coding)

  • 能抄不写，能连不造，能复用不原创
  • 缓解三大缺陷：AI 幻觉、复杂性爆炸、门槛过高

🎨 Canvas 白板驱动开发

  • 让白板成为单一真相源
  • 图形是第一公民，代码是白板的序列化

🐝 AI 蜂群协作

  • 多 AI 在 tmux 下互相感知、协作
  • 人从瓶颈变为调度者
工具资源：
  • Chat Vault - AI 聊天记录保存工具
  • prompts-library - Excel ↔ Markdown 互转工具
  • 备份脚本 - 一键备份和快速备份
核心经验：
  • 人下 AI 上
  • 一切问题问 AI
  • 上下文是 vibe coding 的第一性要素
  • 模仿优先，不重复造轮子
  • 文档即上下文，不是事后补

  • ✅ 掌握方法论，从"会用工具"到"理解原理"
  • ✅ 学习规划驱动、胶水编程等高级实践

前置要求：
