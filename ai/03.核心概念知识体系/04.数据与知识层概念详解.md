## 1. Chunking（分块策略）

### 🎯 掌握程度：必须掌握

### 什么是Chunking？

**通俗解释：**
Chunking就是"把长文章切成小段"。就像把长面包切成片，方便食用。把长文档切成小段，方便AI理解和检索。

**技术定义：**
将长文本分割成适合处理的小块（chunks），是构建向量数据库的核心步骤。

### 为什么需要Chunking？

```
问题：长文档无法直接处理

❌ 整个文档作为一个向量：
├─ 文档太长：超过Token限制
├─ 信息过载：检索时返回过多无关信息
├─ 语义模糊：向量无法准确表示
└─ 存储浪费：每个文档一个向量，空间利用率低

✅ 文档分块后：
├─ 精确检索：只返回相关段落
├─ 语义清晰：每块语义完整
├─ 存储高效：多个向量，灵活检索
└─ 成本降低：只检索需要的部分
```

### Chunking策略对比

#### 策略1：固定大小分块（Fixed-size）

```java
@Service
public class FixedSizeChunker {

    private static final int CHUNK_SIZE = 500;  // 字符数
    private static final int OVERLAP = 50;      // 重叠字符数

    public List<String> chunk(String text) {

        List<String> chunks = new ArrayList<>();
        int start = 0;

        while (start < text.length()) {

            int end = Math.min(start + CHUNK_SIZE, text.length());

            // 提取chunk
            String chunk = text.substring(start, end);
            chunks.add(chunk);

            // 移动到下一个chunk（考虑重叠）
            start = end - OVERLAP;

            // 防止无限循环
            if (start >= text.length()) {
                break;
            }
        }

        return chunks;
    }
}
```

**优点：**
- 实现简单
- 性能好
- 覆盖全面

**缺点：**
- 可能在句子中间截断
- 语义不完整

#### 策略2：段落分块（Paragraph-based）

```java
@Service
public class ParagraphChunker {

    public List<String> chunkByParagraph(String text) {

        // 按段落分割（\n\n 或 \n）
        String[] paragraphs = text.split("\\n\\n|\\n");

        List<String> chunks = new ArrayList<>();
        StringBuilder currentChunk = new StringBuilder();
        int currentLength = 0;
        final int MAX_LENGTH = 1000;

        for (String paragraph : paragraphs) {

            // 如果单个段落太长，需要进一步分割
            if (paragraph.length() > MAX_LENGTH) {
                if (currentChunk.length() > 0) {
                    chunks.add(currentChunk.toString());
                    currentChunk = new StringBuilder();
                    currentLength = 0;
                }

                // 对长段落进行强制分割
                List<String> subChunks = splitLongParagraph(paragraph, MAX_LENGTH);
                chunks.addAll(subChunks);
                continue;
            }

            // 检查是否超出限制
            if (currentLength + paragraph.length() > MAX_LENGTH) {
                if (currentChunk.length() > 0) {
                    chunks.add(currentChunk.toString());
                    currentChunk = new StringBuilder();
                    currentLength = 0;
                }
            }

            // 添加段落
            currentChunk.append(paragraph).append("\n\n");
            currentLength += paragraph.length() + 2;
        }

        // 添加最后一个chunk
        if (currentChunk.length() > 0) {
            chunks.add(currentChunk.toString());
        }

        return chunks;
    }

    private List<String> splitLongParagraph(String paragraph, int maxLength) {

        List<String> chunks = new ArrayList<>();
        int start = 0;

        while (start < paragraph.length()) {

            int end = Math.min(start + maxLength, paragraph.length());

            // 尝试在句子边界分割
            if (end < paragraph.length()) {
                // 往前找最近的句号
                int lastPeriod = paragraph.lastIndexOf("。", end);
                if (lastPeriod > start) {
                    end = lastPeriod + 1;
                }
            }

            chunks.add(paragraph.substring(start, end).trim());
            start = end;
        }

        return chunks;
    }
}
```

**优点：**
- 保持段落完整
- 语义相对完整
- 更易阅读

**缺点：**
- 实现复杂
- 段落长度不均

#### 策略3：语义分块（Semantic）

```java
@Service
public class SemanticChunker {

    @Autowired
    private EmbeddingModel embeddingModel;

    public List<String> chunkBySemantic(String text) {

        // 1. 先按段落粗分
        List<String> paragraphs = Arrays.asList(text.split("\\n\\n"));

        // 2. 计算每段的向量
        List<float[]> embeddings = paragraphs.stream()
            .map(para -> embeddingModel.embed(para).content())
            .collect(Collectors.toList());

        // 3. 计算相邻段落的语义距离
        List<Double> distances = new ArrayList<>();
        for (int i = 0; i < embeddings.size() - 1; i++) {
            double distance = 1 - cosineSimilarity(
                embeddings.get(i),
                embeddings.get(i + 1)
            );
            distances.add(distance);
        }

        // 4. 在语义距离大的地方分割
        List<String> chunks = new ArrayList<>();
        StringBuilder currentChunk = new StringBuilder();

        // 找出语义距离的阈值（如前25%分位数）
        double threshold = calculateThreshold(distances, 0.75);

        for (int i = 0; i < paragraphs.size(); i++) {

            currentChunk.append(paragraphs.get(i)).append("\n\n");

            // 如果是最后一段，或与下一段语义差异大
            if (i == paragraphs.size() - 1 ||
                distances.get(i) > threshold) {

                chunks.add(currentChunk.toString().trim());
                currentChunk = new StringBuilder();
            }
        }

        return chunks;
    }

    private double calculateThreshold(List<Double> distances,
                                     double percentile) {
        // 计算分位数作为阈值
        List<Double> sorted = new ArrayList<>(distances);
        Collections.sort(sorted);

        int index = (int) (sorted.size() * percentile);
        return sorted.get(index);
    }
}
```

**优点：**
- 语义完整性最好
- 检索质量最高

**缺点：**
- 计算成本高（需要多次Embedding）
- 实现复杂

#### 策略4：递归分块（Recursive）

```java
@Service
public class RecursiveChunker {

    // 多级分割符：从粗到细
    private static final String[] SEPARATORS = {
        "\n\n\n",  // 三次换行（章节）
        "\n\n",    // 两次换行（段落）
        "\n",      // 一次换行（行）
        "。",      // 句号（句子）
        "，",      // 逗号（短语）
        " "        // 空格（词）
    };

    public List<String> chunkRecursive(String text,
                                       int chunkSize,
                                       int overlap) {

        return chunkRecursive(text, chunkSize, overlap, 0);
    }

    private List<String> chunkRecursive(String text,
                                       int chunkSize,
                                       int overlap,
                                       int separatorIndex) {

        List<String> chunks = new ArrayList<>();

        // 如果文本足够短，直接返回
        if (text.length() <= chunkSize) {
            chunks.add(text);
            return chunks;
        }

        // 使用当前分割符
        if (separatorIndex < SEPARATORS.length) {

            String separator = SEPARATORS[separatorIndex];
            String[] splits = text.split(Pattern.quote(separator));

            StringBuilder currentChunk = new StringBuilder();

            for (String split : splits) {

                String trimmed = split.trim();

                // 如果单个split太长，递归使用更细的分割符
                if (trimmed.length() > chunkSize) {
                    if (currentChunk.length() > 0) {
                        chunks.add(currentChunk.toString());
                        currentChunk = new StringBuilder();
                    }

                    chunks.addAll(chunkRecursive(
                        trimmed,
                        chunkSize,
                        overlap,
                        separatorIndex + 1
                    ));
                    continue;
                }

                // 检查是否超出chunk大小
                if (currentChunk.length() + trimmed.length() + separator.length()
                    > chunkSize) {

                    if (currentChunk.length() > 0) {
                        chunks.add(currentChunk.toString());
                        currentChunk = new StringBuilder();
                    }
                }

                // 添加到当前chunk
                if (currentChunk.length() > 0) {
                    currentChunk.append(separator);
                }
                currentChunk.append(trimmed);
            }

            // 添加最后一个chunk
            if (currentChunk.length() > 0) {
                chunks.add(currentChunk.toString());
            }

        } else {
            // 没有更细的分割符，强制分割
            chunks.addAll(forceChunk(text, chunkSize, overlap));
        }

        return chunks;
    }

    private List<String> forceChunk(String text,
                                    int chunkSize,
                                    int overlap) {

        List<String> chunks = new ArrayList<>();
        int start = 0;

        while (start < text.length()) {
            int end = Math.min(start + chunkSize, text.length());
            chunks.add(text.substring(start, end));
            start = end - overlap;
        }

        return chunks;
    }
}
```

**优点：**
- 灵活性高
- 保持语义结构
- 适用各种文本

**缺点：**
- 实现复杂
- 需要调参

### 最佳实践

```java
@Service
public class ChunkingBestPractices {

    // 推荐配置
    private static final int OPTIMAL_CHUNK_SIZE = 500;  // 500-1000字符
    private static final int OPTIMAL_OVERLAP = 50;      // 10-20%重叠

    // 实际应用：综合策略
    public List<Chunk> smartChunk(String document) {

        // 1. 预处理
        String cleaned = cleanText(document);

        // 2. 按结构分割（章节、段落）
        List<String> structuralChunks = splitByStructure(cleaned);

        // 3. 每个结构块内部递归分块
        List<Chunk> chunks = new ArrayList<>();
        for (String structuralChunk : structuralChunks) {

            List<String> subChunks = chunkRecursive(
                structuralChunk,
                OPTIMAL_CHUNK_SIZE,
                OPTIMAL_OVERLAP
            );

            // 4. 为每个chunk添加元数据
            for (int i = 0; i < subChunks.size(); i++) {
                chunks.add(new Chunk(
                    subChunks.get(i),
                    buildMetadata(structuralChunk, i, subChunks.size())
                ));
            }
        }

        return chunks;
    }

    private String cleanText(String text) {
        // 移除多余空白
        text = text.replaceAll("\\s+", " ");

        // 修复乱码
        text = fixEncoding(text);

        // 统一换行符
        text = text.replaceAll("\\r\\n|\\r", "\n");

        return text;
    }

    private Map<String, Object> buildMetadata(String source,
                                              int index,
                                              int total) {

        Map<String, Object> metadata = new HashMap<>();
        metadata.put("source", source);
        metadata.put("chunk_index", index);
        metadata.put("total_chunks", total);
        metadata.put("timestamp", Instant.now());

        return metadata;
    }
}
```

### Chunk Size选择

```
推荐配置（经验值）：

短文本（< 1000字）：
├─ Chunk Size: 200-300 字符
├─ Overlap: 20-50 字符
└─ 场景：FAQ、对话

中等文本（1000-5000字）：
├─ Chunk Size: 500-800 字符
├─ Overlap: 50-100 字符
└─ 场景：文章、报告

长文本（> 5000字）：
├─ Chunk Size: 800-1000 字符
├─ Overlap: 100-150 字符
└─ 场景：书籍、文档

技术文档：
├─ Chunk Size: 按代码块分割
├─ Overlap: 保留上下文
└─ 场景：API文档、代码
```

## 2. Data Cleaning（数据清洗）

### 🎯 掌握程度：必须掌握

### 什么是数据清洗？

**通俗解释：**
数据清洗就像"打扫房间"。把脏乱的数据整理干净，让知识库更整洁、更准确。

**技术定义：**
去除数据中的噪声、错误、重复等内容，提升数据质量的过程。

### 为什么需要数据清洗？

```
脏数据的问题：
❌ 影响检索质量
   "如何    使用    Java？"  → 多余空格

❌ 影响向量化
   "[乱码内容]使用说明" → 向量不准确

❌ 影响用户体验
   "详见附表一（见下页）" → 脱离上下文，无意义

清洗后的效果：
✅ 检索更准确
✅ 向量更精准
✅ 用户体验更好
```

### 数据清洗技术

#### 技术1：文本规范化

```java
@Service
public class TextNormalizer {

    public String normalize(String text) {

        // 1. 统一换行符
        text = text.replaceAll("\\r\\n|\\r", "\n");

        // 2. 移除多余空白
        text = text.replaceAll("\\s+", " ");

        // 3. 移除控制字符
        text = text.replaceAll("[\\p{Cntrl}]", "");

        // 4. 统一引号
        text = text.replaceAll("[\u201c\u201d]", "\"");  // 中文引号→英文
        text = text.replaceAll("[\u2018\u2019]", "'");  // 中文单引号→英文

        // 5. 全角转半角
        text = fullWidthToHalfWidth(text);

        return text;
    }

    private String fullWidthToHalfWidth(String text) {
        StringBuilder result = new StringBuilder();

        for (char c : text.toCharArray()) {
            if (c >= 0xFF01 && c <= 0xFF5E) {
                // 全角字符转半角
                result.append((char) (c - 0xFEE0));
            } else if (c == 0x3000) {
                // 全角空格转半角
                result.append(' ');
            } else {
                result.append(c);
            }
        }

        return result.toString();
    }
}
```

#### 技术2：去除无用信息

```java
@Service
public class ContentCleaner {

    public String clean(String text) {

        // 1. 移除页眉页脚
        text = removeHeadersAndFooters(text);

        // 2. 移除页码
        text = removePageNumbers(text);

        // 3. 移除特殊标记
        text = removeSpecialMarkers(text);

        // 4. 移除HTML标签
        text = removeHtmlTags(text);

        return text;
    }

    private String removeHeadersAndFooters(String text) {

        // 移除常见的页眉页脚模式
        String[] patterns = {
            "第.*页.*共.*页",
            "Page \\d+ of \\d+",
            "保密文件.*内部资料",
            "^\\s*.{0,20}\\s*$"  // 过短的行（可能是页眉页脚）
        };

        for (String pattern : patterns) {
            text = text.replaceAll(pattern, "");
        }

        return text;
    }

    private String removePageNumbers(String text) {

        // 移除单独成行的页码
        text = text.replaceAll("^\\s*-?\\d+-?\\s*$", "");

        return text;
    }

    private String removeSpecialMarkers(String text) {

        // 移除HTML注释
        text = text.replaceAll("<!--.*?-->", "");

        // 移除Markdown特殊符号（保留内容）
        text = text.replaceAll("\\[\\^.*?\\]", "");  // 脚注标记
        text = text.replaceAll("\\[.*?\\]\\(.*?\\)", "$2");  // Markdown链接

        return text;
    }

    private String removeHtmlTags(String text) {

        // 简单的HTML标签移除
        text = text.replaceAll("<[^>]+>", "");

        return text;
    }
}
```

#### 技术3：去重

```java
@Service
public class DocumentDeduplicator {

    @Autowired
    private EmbeddingModel embeddingModel;

    @Autowired
    private VectorStore vectorStore;

    public List<String> deduplicate(List<String> documents) {

        List<String> uniqueDocs = new ArrayList<>();
        Set<String> seenHashes = new HashSet<>();

        for (String doc : documents) {

            // 1. 计算哈希（精确去重）
            String hash = DigestUtils.md5DigestAsHex(
                doc.getBytes(StandardCharsets.UTF_8)
            );

            if (seenHashes.contains(hash)) {
                continue;  // 完全重复，跳过
            }

            // 2. 语义去重
            if (isSemanticallyDuplicate(doc, uniqueDocs)) {
                continue;  // 语义重复，跳过
            }

            uniqueDocs.add(doc);
            seenHashes.add(hash);
        }

        return uniqueDocs;
    }

    private boolean isSemanticallyDuplicate(String doc,
                                           List<String> existing) {

        // 向量化当前文档
        float[] docEmbedding = embeddingModel.embed(doc).content();

        // 与已有文档比较
        for (String existingDoc : existing) {

            float[] existingEmbedding = embeddingModel
                .embed(existingDoc).content();

            double similarity = cosineSimilarity(
                docEmbedding,
                existingEmbedding
            );

            // 如果相似度 > 0.95，认为是重复
            if (similarity > 0.95) {
                return true;
            }
        }

        return false;
    }
}
```

#### 技术4：敏感信息脱敏

```java
@Service
public class PIIRedactor {

    // PII模式
    private static final Pattern EMAIL_PATTERN =
        Pattern.compile("\\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\\.[A-Z|a-z]{2,}\\b");

    private static final Pattern PHONE_PATTERN =
        Pattern.compile("(?:(?:\\+|00)86)?1[3-9]\\d{9}");

    private static final Pattern ID_CARD_PATTERN =
        Pattern.compile("\\d{17}[\\dXx]");

    private static final Pattern CREDIT_CARD_PATTERN =
        Pattern.compile("\\b(?:\\d[ -]*?){13,16}\\b");

    public String redact(String text) {

        // 1. 邮箱脱敏
        text = EMAIL_PATTERN.matcher(text)
            .replaceAll(m -> {
                String email = m.group();
                String[] parts = email.split("@");
                String name = parts[0];
                String domain = parts[1];

                // 只保留首尾
                String maskedName = name.charAt(0) +
                    "*****" +
                    name.charAt(name.length() - 1);

                return maskedName + "@" + domain;
            });

        // 2. 手机号脱敏
        text = PHONE_PATTERN.matcher(text)
            .replaceAll(m -> {
                String phone = m.group();
                return phone.substring(0, 3) + "****" +
                    phone.substring(7);
            });

        // 3. 身份证脱敏
        text = ID_CARD_PATTERN.matcher(text)
            .replaceAll("$1**********$2");

        // 4. 银行卡脱敏
        text = CREDIT_CARD_PATTERN.matcher(text)
            .replaceAll(m -> {
                String card = m.group().replaceAll("[ -]", "");
                if (card.length() >= 16) {
                    return card.substring(0, 4) + "********" +
                        card.substring(card.length() - 4);
                }
                return m.group();
            });

        return text;
    }

    // 使用NER模型识别更多PII
    public String advancedRedact(String text) {

        // 调用NER服务识别实体
        List<PIIEntity> entities = nerService.detectPII(text);

        // 按位置倒序替换（避免索引偏移）
        entities.sort((a, b) ->
            Integer.compare(b.getStart(), a.getStart()));

        StringBuilder sb = new StringBuilder(text);

        for (PIIEntity entity : entities) {

            String replacement = switch (entity.getType()) {
                case "PERSON" -> maskName(entity.getText());
                case "ADDRESS" -> "[地址已隐藏]";
                case "ORG" -> entity.getText();  // 组织名称不脱敏
                default -> entity.getText();
            };

            sb.replace(entity.getStart(), entity.getEnd(), replacement);
        }

        return sb.toString();
    }

    private String maskName(String name) {

        if (name.length() <= 2) {
            return "*".repeat(name.length());
        }

        return name.charAt(0) + "*".repeat(name.length() - 2) +
            name.charAt(name.length() - 1);
    }
}
```

### 数据清洗Pipeline

```java
@Service
public class DataCleaningPipeline {

    @Autowired
    private TextNormalizer normalizer;

    @Autowired
    private ContentCleaner cleaner;

    @Autowired
    private DocumentDeduplicator deduplicator;

    @Autowired
    private PIIRedactor piiRedactor;

    public List<CleanDocument> clean(List<RawDocument> rawDocuments) {

        log.info("Starting data cleaning pipeline for {} documents",
            rawDocuments.size());

        // Stage 1: 提取文本
        List<String> extractedTexts = rawDocuments.stream()
            .map(this::extractText)
            .collect(Collectors.toList());

        // Stage 2: 规范化
        List<String> normalized = extractedTexts.stream()
            .map(normalizer::normalize)
            .collect(Collectors.toList());

        // Stage 3: 清洗
        List<String> cleaned = normalized.stream()
            .map(cleaner::clean)
            .collect(Collectors.toList());

        // Stage 4: 脱敏
        List<String> redacted = cleaned.stream()
            .map(piiRedactor::redact)
            .collect(Collectors.toList());

        // Stage 5: 去重
        List<String> deduplicated = deduplicator.deduplicate(redacted);

        // Stage 6: 质量检查
        List<CleanDocument> finalDocs = deduplicated.stream()
            .filter(this::qualityCheck)
            .map(text -> new CleanDocument(text, buildMetadata(text)))
            .collect(Collectors.toList());

        log.info("Data cleaning completed. {} documents passed quality check",
            finalDocs.size());

        return finalDocs;
    }

    private boolean qualityCheck(String text) {

        // 检查长度
        if (text.length() < 50) {
            log.warn("Document too short: {} chars", text.length());
            return false;
        }

        // 检查是否包含过多特殊字符
        long specialCharCount = text.chars()
            .filter(c -> !Character.isLetterOrDigit(c) &&
                       !Character.isWhitespace(c))
            .count();

        if (specialCharCount > text.length() * 0.3) {
            log.warn("Document contains too many special characters");
            return false;
        }

        // 检查是否包含有效内容（非空、非乱码）
        if (!containsValidContent(text)) {
            log.warn("Document appears to be garbage");
            return false;
        }

        return true;
    }

    private boolean containsValidContent(String text) {

        // 检查中文字符比例
        long chineseCharCount = text.chars()
            .filter(c -> c >= 0x4E00 && c <= 0x9FFF)
            .count();

        double chineseRatio = (double) chineseCharCount / text.length();

        // 中文文档：中文字符应占30%以上
        if (chineseRatio > 0.1 && chineseRatio < 0.3) {
            return false;
        }

        return true;
    }
```

```java
@Service
public class MetadataExtractor {

    // 从文档提取元数据
    public DocumentMetadata extract(Document doc) {

        DocumentMetadata.Builder builder = DocumentMetadata.builder();

        // 1. 基础信息
        builder.documentId(UUID.randomUUID().toString());
        builder.source(extractSource(doc));
        builder.createdAt(Instant.now());

        // 2. 从文档内容提取标题
        builder.title(extractTitle(doc.getContent()));

        // 3. 自动分类
        builder.category(classifyDocument(doc.getContent()));

        // 4. 提取关键词
        builder.tags(extractKeywords(doc.getContent()));

        // 5. 语言检测
        builder.language(detectLanguage(doc.getContent()));

        // 6. 访问级别判断
        builder.accessLevel(determineAccessLevel(doc.getContent()));

        // 7. 内容类型判断
        builder.contentType(detectContentType(doc.getContent()));

        return builder.build();
    }

    private String extractTitle(String content) {

        // 尝试提取第一行作为标题
        String[] lines = content.split("\\n", 2);

        if (lines.length > 0) {
            String firstLine = lines[0].trim();

            // 标题通常较短（< 100字符）
            if (firstLine.length() < 100) {
                return firstLine;
            }
        }

        // 如果没有明显标题，使用"未命名文档"
        return "未命名文档";
    }

    private String classifyDocument(String content) {

        String prompt = String.format("""
            请将以下文档分类到以下类别之一：
            - 技术文档
            - 产品说明
            - 操作指南
            - FAQ
            - 政策制度
            - 其他

            文档内容（前500字）：
            %s

            只返回类别名称。
            """, content.substring(0, Math.min(500, content.length()))
        );

        String category = llmService.chat(prompt);
        return category.trim();
    }

    private List<String> extractKeywords(String content) {

        // 使用LLM提取关键词
        String prompt = String.format("""
            从以下文档中提取5-10个关键词标签：

            %s

            返回格式：标签1, 标签2, 标签3, ...
            """, content.substring(0, Math.min(1000, content.length()))
        );

        String response = llmService.chat(prompt);

        return Arrays.stream(response.split(","))
            .map(String::trim)
            .collect(Collectors.toList());
    }

    private String detectLanguage(String content) {

        // 简单的中文检测
        long chineseChars = content.chars()
            .filter(c -> c >= 0x4E00 && c <= 0x9FFF)
            .count();

        if (chineseChars > content.length() * 0.3) {
            return "zh-CN";
        } else {
            return "en-US";
        }
    }

    private String determineAccessLevel(String content) {

        // 检测敏感词
        if (containsSensitiveKeywords(content)) {
            return "confidential";
        }

        // 检测内部标记
        if (content.contains("内部") || content.contains("保密")) {
            return "internal";
        }

        return "public";
    }

    private boolean containsSensitiveKeywords(String content) {

        String[] sensitiveKeywords = {
            "机密", "秘密", "涉密",
            "Confidential", "Secret"
        };

        for (String keyword : sensitiveKeywords) {
            if (content.contains(keyword)) {
                return true;
            }
        }

        return false;
    }
}
```

```java
@Service
public class MetadataAwareSearch {

    @Autowired
    private VectorStore vectorStore;

    public List<Document> searchWithMetadata(
        String query,
        Map<String, Object> filters
    ) {

        // 1. 问题向量化
        float[] queryEmbedding = embeddingModel.embed(query).content();

        // 2. 基础向量检索
        List<EmbeddingMatch<Document>> matches =
            vectorStore.findRelevant(queryEmbedding, 50);  // 召回50个

        // 3. 应用元数据过滤
        List<Document> filtered = matches.stream()
            .map(EmbeddingMatch::embedded)
            .filter(doc -> matchesFilters(doc.getMetadata(), filters))
            .collect(Collectors.toList());

        // 4. 重新排序（结合元数据）
        return rerankWithMetadata(filtered, query, filters);
    }

    private boolean matchesFilters(DocumentMetadata metadata,
                                   Map<String, Object> filters) {

        for (Map.Entry<String, Object> filter : filters.entrySet()) {

            String key = filter.getKey();
            Object value = filter.getValue();

            switch (key) {
                case "category":
                    if (!metadata.getCategory().equals(value)) {
                        return false;
                    }
                    break;

                case "tags":
                    List<String> requiredTags = (List<String>) value;
                    if (!metadata.getTags().containsAll(requiredTags)) {
                        return false;
                    }
                    break;

                case "accessLevel":
                    String required = (String) value;
                    String allowed = metadata.getAccessLevel();

                    // accessLevel优先级：confidential > internal > public
                    if (getAccessLevelPriority(allowed) <
                        getAccessLevelPriority(required)) {
                        return false;
                    }
                    break;

                case "dateRange":
                    Map<String, LocalDate> range =
                        (Map<String, LocalDate>) value;
                    LocalDate updated = metadata.getUpdatedAt().toLocalDate();

                    if (updated.isBefore(range.get("start")) ||
                        updated.isAfter(range.get("end"))) {
                        return false;
                    }
                    break;
            }
        }

        return true;
    }

    private List<Document> rerankWithMetadata(
        List<Document> documents,
        String query,
        Map<String, Object> filters
    ) {

        // 结合向量相似度和元数据进行重排序
        return documents.stream()
            .sorted((d1, d2) -> {

                double score1 = calculateCompositeScore(
                    d1,
                    query,
                    filters
                );

                double score2 = calculateCompositeScore(
                    d2,
                    query,
                    filters
                );

                return Double.compare(score2, score1);  // 降序
            })
            .collect(Collectors.toList());
    }

    private double calculateCompositeScore(
        Document doc,
        String query,
        Map<String, Object> filters
    ) {

        DocumentMetadata metadata = doc.getMetadata();

        // 基础向量相似度
        double baseScore = doc.getSimilarity();

        // 质量分数加成
        double qualityBoost = metadata.getQualityScore() * 0.1;

        // 新鲜度加成（最近文档优先）
        long daysSinceUpdate = ChronoUnit.DAYS.between(
            metadata.getUpdatedAt(),
            Instant.now()
        );
        double freshnessBoost = Math.max(0, 1 - daysSinceUpdate / 365.0) * 0.05;

        // 热门度加成
        double popularityBoost = Math.log1d(metadata.getViewCount()) * 0.02;

        return baseScore + qualityBoost + freshnessBoost + popularityBoost;
    }

    private int getAccessLevelPriority(String level) {

        return switch (level) {
            case "public" -> 1;
            case "internal" -> 2;
            case "confidential" -> 3;
            default -> 0;
        };
    }
}
```

## 4. Re-ranking（重排序）

### 📖 掌握程度：需要了解

### 什么是Re-ranking？

**通俗解释：**
重排序就像"精选"。先粗选出候选答案，再仔细比较，选出最准确的几个。

**技术定义：**
对初步检索结果进行精细化排序的过程，使用更精确但更慢的模型。

### 为什么需要Re-ranking？

```
问题：向量检索不够精确

查询："如何修复数据库连接超时？"

向量检索结果（Top 5）：
1. "数据库连接池配置"（相似度0.85） ← 相关，但不直接
2. "网络超时问题排查"（相似度0.83） ← 相关
3. "数据库连接超时修复方法"（相似度0.82） ← 最相关！
4. "数据库性能优化"（相似度0.80） ← 不太相关
5. "JDBC驱动配置"（相似度0.79） ← 相关

重排序后（基于更精确的模型）：
1. "数据库连接超时修复方法" ← 正确排到第一
2. "网络超时问题排查"
3. "数据库连接池配置"
...
```

### Re-ranking实现

```java
@Service
public class ReRankingService {

    // 使用Cross-Encoder进行重排序
    public List<Document> rerank(String query,
                                 List<Document> candidates) {

        // 1. 为每个候选文档计算精确相关性分数
        List<ScoredDocument> scored = candidates.stream()
            .map(doc -> {
                double score = calculateRelevanceScore(query, doc);
                return new ScoredDocument(doc, score);
            })
            .collect(Collectors.toList());

        // 2. 重新排序
        scored.sort((a, b) -> Double.compare(b.score, a.score));

        // 3. 返回Top N
        return scored.stream()
            .limit(10)
            .map(ScoredDocument::getDocument)
            .collect(Collectors.toList());
    }

    private double calculateRelevanceScore(String query, Document doc) {

        // 方法1：使用LLM直接评分
        return llmRelevanceScore(query, doc);

        // 方法2：使用专门的Reranker模型
        // return crossEncoder.score(query, doc.getContent());
    }

    private double llmRelevanceScore(String query, Document doc) {

        String prompt = String.format("""
            请评估以下文档与查询的相关性（0-1分）：

            查询：%s

            文档：%s

            只返回一个数字（0-1之间的小数）。
            """, query, doc.getContent().substring(0, 500)
        );

        String response = llmService.chat(prompt);

        try {
            return Double.parseDouble(response.trim());
        } catch (NumberFormatException e) {
            log.warn("Failed to parse relevance score: {}", response);
            return 0.5;
        }
    }
}
```

```java
    private VectorStore vectorStore;

    private KeywordSearchService keywordSearch;

    private ReRankingService reranker;

    public List<Document> search(String query) {

        CompletableFuture<List<Document>> vectorFuture =
            CompletableFuture.supplyAsync(() -> vectorSearch(query));

        CompletableFuture<List<Document>> keywordFuture =
            CompletableFuture.supplyAsync(() -> keywordSearch.search(query));

        // 等待两种检索完成
        CompletableFuture.allOf(vectorFuture, keywordFuture).join();

        List<Document> combined = new ArrayList<>();
        combined.addAll(vectorFuture.join());
        combined.addAll(keywordFuture.join());

        // 去重
        List<Document> deduplicated = deduplicate(combined);

        // Stage 3: Re-ranking
        return reranker.rerank(query, deduplicated);
    }
}
```

## 5. Hybrid Search（混合检索）

### 🚀 掌握程度：进阶学习

### 什么是混合检索？

**通俗解释：**
混合检索就是"多种方法一起找"。同时用向量检索（理解语义）和关键词检索（精确匹配），结果合并后更准确。

**技术定义：**
结合向量检索和关键词检索的搜索方法，取两者之长。

### 混合检索实现

```java
@Service
public class HybridSearchService {

    @Autowired
    private VectorStore vectorStore;

    @Autowired
    private KeywordSearchService keywordSearch;

    public List<Document> hybridSearch(String query,
                                      int alpha) {
        // alpha: 向量检索的权重（0-1）
        // 1-alpha: 关键词检索的权重

        // 1. 向量检索
        List<SearchResult> vectorResults = vectorStore
            .search(query, 50)
            .stream()
            .map(doc -> new SearchResult(doc, doc.getSimilarity()))
            .collect(Collectors.toList());

        // 2. 关键词检索（BM25）
        List<SearchResult> keywordResults = keywordSearch
            .searchBM25(query, 50)
            .stream()
            .map(result -> new SearchResult(
                result.getDocument(),
                result.getBm25Score()
            ))
            .collect(Collectors.toList());

        // 3. 分数归一化
        normalizeScores(vectorResults);
        normalizeScores(keywordResults);

        // 4. 合并分数
        Map<String, SearchResult> merged = new HashMap<>();

        for (SearchResult result : vectorResults) {
            String docId = result.getDocument().getId();
            merged.putIfAbsent(docId, result);
            merged.get(docId).addVectorScore(result.getScore());
        }

        for (SearchResult result : keywordResults) {
            String docId = result.getDocument().getId();
            merged.putIfAbsent(docId, result);
            merged.get(docId).addKeywordScore(result.getScore());
        }

        // 5. 计算混合分数
        for (SearchResult result : merged.values()) {
            double hybridScore = alpha * result.getVectorScore() +
                                (1 - alpha) * result.getKeywordScore();
            result.setScore(hybridScore);
        }

        // 6. 排序并返回Top N
        return merged.values().stream()
            .sorted((a, b) -> Double.compare(b.getScore(), a.getScore()))
            .limit(20)
            .map(SearchResult::getDocument)
            .collect(Collectors.toList());
    }

    private void normalizeScores(List<SearchResult> results) {

        if (results.isEmpty()) return;

        // Min-Max归一化到[0, 1]
        double min = results.stream()
            .mapToDouble(SearchResult::getScore)
            .min()
            .orElse(0);

        double max = results.stream()
            .mapToDouble(SearchResult::getScore)
            .max()
            .orElse(1);

        double range = max - min;

        for (SearchResult result : results) {
            double normalized = range == 0 ?
                0 :
                (result.getScore() - min) / range;
            result.setScore(normalized);
        }
    }
}
```

```java
@Service
public class KnowledgeBaseBuilder {

    @Autowired
    private DocumentExtractor extractor;

    @Autowired
    private DataCleaningPipeline cleaningPipeline;

    @Autowired
    private SmartChunker chunker;

    @Autowired
    private EmbeddingService embeddingService;

    @Autowired
    private VectorStore vectorStore;

    @Autowired
    private MetadataExtractor metadataExtractor;

    @Async
    public void buildKnowledgeBase(List<SourceDocument> sources) {

        log.info("Building knowledge base from {} sources", sources.size());

        int totalDocuments = 0;
        int totalChunks = 0;

        for (SourceDocument source : sources) {

            try {
                // Stage 1: 提取文本
                String text = extractor.extract(source);
                log.debug("Extracted {} chars from {}", text.length(), source);

                // Stage 2: 清洗
                CleanDocument cleaned = cleaningPipeline.clean(
                    List.of(new RawDocument(text, source))
                ).get(0);

                // Stage 3: 分块
                List<Chunk> chunks = chunker.smartChunk(cleaned.getText());
                log.debug("Split into {} chunks", chunks.size());

                // Stage 4: 处理每个chunk
                for (Chunk chunk : chunks) {

                    // 提取元数据
                    DocumentMetadata metadata = metadataExtractor.extract(
                        new Document(chunk.getText(), source)
                    );

                    // 向量化
                    float[] embedding = embeddingService.embed(chunk.getText());

                    // 存储
                    vectorStore.insert(new VectorDocument(
                        chunk.getText(),
                        embedding,
                        metadata
                    ));

                    totalChunks++;
                }

                totalDocuments++;

            } catch (Exception e) {
                log.error("Failed to process document: {}", source, e);
            }
        }

        log.info("Knowledge base built successfully: {} documents, {} chunks",
            totalDocuments, totalChunks);
    }

    // 增量更新
    public void updateKnowledgeBase(List<SourceDocument> newSources) {

        for (SourceDocument source : newSources) {

            // 检查是否已存在
            if (vectorStore.exists(source.getId())) {

                // 删除旧版本
                vectorStore.deleteBySourceId(source.getId());
            }

            // 处理新版本
            buildKnowledgeBase(List.of(source));
        }
    }

    // 质量检查
    public QualityReport qualityCheck() {

        // 测试查询集
        List<TestQuery> testQueries = loadTestQueries();

        int total = testQueries.size();
        int passed = 0;

        for (TestQuery test : testQueries) {

            List<Document> results = vectorStore.search(
                test.getQuery(),
                5
            );

            if (isRelevant(results, test.getExpectedDocId())) {
                passed++;
            }
        }

        double accuracy = (double) passed / total;

        return new QualityReport(total, passed, accuracy);
    }
}
```
